# Gemzi Boutique - Premium AR Jewellery Experience

Gemzi is a state-of-the-art Flutter application designed for high-end jewellery retail, featuring real-time AR try-ons and a digital gold investment ecosystem.

## 🚀 Recent Rebranding & Feature Enhancements

### 1. Unified Brand Identity: "Deep Forest & Gold"
We have finalized and refined the boutique's visual language, moving away from experimental palettes to a signature, high-contrast luxury theme.
- **Primary Palette**: Deep Forest Green (`#0F2F2B`) with Dark Teal (`#17453F`) surfaces.
- **Branding Accents**: Vibrant Gold (`#D4AF37`) applied to all action buttons, highlights, and interactive cues.
- **Typography**: Synchronized **Page Titles** and **Navigation Arrows** across the entire app to the gold brand color for a unified, premium feel.

### 2. AR Try-On Optimization (Face Mesh)
The AR experience has been streamlined to focus on realism and performance.
- **Jewellery Categories**: Optimized for **Earrings** and **Necklaces** using high-precision face-mesh detection.
- **Stability**: Removed obsolete hand-tracking infrastructure to eliminate crashes and ensure a smooth, face-centric boutique experience.
- **Realistic Rendering**: Refined the `OverlayRenderer` to handle real-world lighting and positioning for virtual assets.

### 3. Digital Gold Vault & Wallet
The standard payment interface has been transformed into a sophisticated **Digital Gold Vault**.
- **Visual Vault**: Implemented a "Safe-Style" UI with gradients and secure-lock branding.
- **Animatory Stashing**: Added a dynamic demo where gold coins "drop" into a glassy safe container based on the user's balance.
- **Live Market Sync**: Real-time integration with gold market rates (24K/22K) for instant asset valuation.

### 4. UI/UX Refinement
- **Homepage Curation**: Restructured the boutique dashboard with dedicated sections for "Exclusive Collections" and a "Live Gold Rate" tracker.
- **Seamless Navigation**: Standardized icons and typography across Splash, Settings, and Shop pages.
- **Interactive Feedback**: Integrated `AnimateDo` for smooth transitions and gold-themed micro-animations.

### 5. Technical Stability
- **Compilation Fixes**: Resolved all "Not a constant expression" errors and identifier conflicts caused by global theme migrations.
- **Camera Lifecycle**: Optimized camera initialization for faster AR start times.
- **Resource Management**: Purged legacy assets and hand-detection logic to reduce app size and improve runtime efficiency.
